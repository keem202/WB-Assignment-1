# WB-Assignment – Interactive CV Webpage

This repository contains my personal CV webpage created for the **Design of Web-Based Systems** course.
The webpage is built using **HTML, CSS, and JavaScript** and includes interactive features such as dark mode, form validation, and dynamic content.

## View My CV Online

You can view the live webpage here:

**https://keem202.github.io/WB-Assignment-1/**

## Project Structure

```
index.html      → Main CV webpage
styles.css      → Styling for the webpage
script.js       → JavaScript interactivity
images/         → Profile image and assets
link.txt        → Public URL of the published webpage
```

## Features

* Dark Mode Toggle
* Skills Show/Hide Interaction
* Contact Form with JavaScript Validation
* Welcome Message on Page Load

## Technologies Used

* HTML
* CSS
* JavaScript
* GitHub Pages (for hosting)
